# App de Receitas :fork_and_knife: :cocktail:

Neste projeto desenvolvido em grupo o objetivo foi colocar em prática tudo o que aprendemos sobre React Context API e React Hooks. Praticamos a organização de um projeto em equipe com a metodologia agile Kanban.

Desenvolvemos um app de receitas que é conectado a duas 2 API’s ao mesmo tempo! 🎒

Além disso, esse é um projeto interface mobile only 📱.

<details>
  <summary><strong>👨‍💻 O que foi desenvolvido</strong></summary><br />

 Desenvolvemos um app de receitas utilizando o que há de mais moderno dentro do ecossistema React: Hooks e Context API!

  Nele, é possível: ver, buscar, filtrar, favoritar e acompanhar o progresso de preparação de receitas de comidas e bebidas!

  ⚠️ A base de dados serão duas APIs distintas: uma para comidas e outra para bebidas.
</details>

<details>
  <summary><strong>:memo: Habilidades desenvolvidas</strong></summary><br />

  Neste projeto, aprendi:

  - Utilizar _Redux_ para gerenciar estado.
  - Utilizar a biblioteca _React-Redux_.
  - Utilizar a Context API do _React_ para gerenciar estado.
  - Utilizar o _React Hook useState_.
  - Utilizar o _React Hook useContext_.
  - Utilizar o _React Hook useEffect_.
  - Criar Hooks customizados.
</details>
